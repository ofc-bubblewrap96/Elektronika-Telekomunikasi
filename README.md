# Elektronika-Telekomunikasi
Desain dan Simulasi Band Pass Filter Chebyshev 1.8 GHz dengan Bandwidth 20 MHz untuk Aplikasi BTS LTE Berbasis Coupled Microstrip Resonator

Telah dirancang Band Pass Filter (BPF) pada frekuensi 1.8 GHz dengan bandwidth 20 MHz untuk aplikasi Base Transceiver Station (BTS) LTE. Filter menggunakan metode Chebyshev order 5 dengan ripple 0.5 dB yang diimplementasikan menggunakan teknologi coupled microstrip resonator pada substrat FR-4. Simulasi dilakukan menggunakan MATLAB untuk menghasilkan dimensi fisik filter berupa lebar jalur (W) = 2.77 mm, panjang resonator (L) = 23 mm, dan spacing antar resonator. Hasil simulasi menunjukkan insertion loss < 0.5 dB dan return loss > 15 dB pada passband 1.79-1.81 GHz.
